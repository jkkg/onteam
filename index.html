PHP

PHP - это распространенный язык программирования общего назначения с открытым исходным кодом для ведения Web-разработок.

Javascript

JavaScript — мультипарадигменный язык программирования, используется как встраиваемый язык для программного доступа к объектам приложений

Symfony

Symfony — свободный фреймворк, написанный на PHP, который использует паттерн Model-View-Controller, позволяет легко решать рутинные задачи веб-программиста.

NodeJS

Node.js добавляет возможность JavaScript взаимодействовать с устройствами ввода-вывода через свой API (написанный на C++), подключать другие внешние библиотеки, написанные на разных языках, обеспечивая вызовы к ним из JavaScript-кода.

MySQL

MySQL — свободная реляционная система управления базами данных. Обычно MySQL используется в качестве сервера, к которому обращаются локальные или удалённые клиенты.
